**担当**: 小林誠（バックエンドミドル）

```python
import sys
import io

def setup_encoding():
    """標準出力のエンコーディングをUTF-8に設定"""
    if sys.platform == 'win32':
        sys.stdout = io.TextIOWrapper(
            sys.stdout.buffer, 
            encoding='utf-8'
        )

def main():
    setup_encoding()
    print("こんにちは")
    return 0

if __name__ == "__main__":
    sys.exit(main())
```
**完了条件**: Windows対応コード追加
