# 🚀 Claude Code 階層型エージェント開発システム v8.8

**企業組織型AIエージェント**による次世代統合開発フレームワーク with 日本標準時統一対応

## 🎯 概要

階層型エージェントシステムは、企業組織を模したAIエージェント群により、あらゆる技術スタックに対応する統合開発環境を提供します。SDD（仕様書駆動開発）とTDD（テスト駆動開発）を融合し、新規開発から既存システム解析まで包括的にサポートします。

## 🏢 システム構成

### 統合開発システム
`.claude_sub_agent/` - 階層型エージェントシステム（全技術スタック統合版）

**統合された技術スタック:**
- .NET Framework (4.0/4.8) + Entity Framework
- Python + FastAPI + SQLAlchemy 2.0
- Vue.js 3 (Axios/Hybrid/Supabase対応)
- VB6レガシー解析・移行
- ISP-673 OCR統合
- SignalR/WebRTC リアルタイム通信

## 🎯 階層型エージェントシステムの特徴

### 組織構造
```
ユーザー
   │
   └─── @cto (プロジェクト担当役員) ← ユーザー唯一の窓口
        │
        ├─── 🛡️ 品質保証部（テスト・品質管理）
        ├─── 🏢 人事部（チーム編成）
        ├─── 💡 経営企画部（戦略立案）
        └─── 💻 システム開発部（実装）
```

### 主要機能
- **解析キャッシュシステム**: 2回目以降70-80%時間削減
- **MCP連携**: Gemini-CLI、o3 MCPとの自動連携
- **即座チーム編成**: AI強化メンバーの自動配属
- **SDD+TDD統合**: 仕様書駆動とテスト駆動の融合
- **マルチエージェントコードレビュー**: 3人の専門レビュアー体制

## 🛠️ 対応技術スタック

### バックエンド
